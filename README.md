# Hartwell Pet Daycare & Boarding

Professional pet daycare, boarding, and grooming services for active, playful dogs and pets.

## About

Hartwell Pet Daycare provides a safe, fun environment for your furry family members. Our facility features a private dog park, supervised play areas, and comfortable boarding accommodations.

## Tech Stack

- **HTML5** with **Tailwind CSS** for responsive design
- **Fredoka** & **Nunito** typography
- **PhotoSwipe** lightbox gallery
- Mobile-first, single-page layout

## Features

- Responsive landing page with sticky navigation
- Service packages (daycare, boarding, grooming)
- Interactive photo gallery with lightbox and progressive loading
- Online booking form
- Contact section with social media integration
- FAQ accordion

## Getting Started

Open `index.html` in your browser — no build step required.
